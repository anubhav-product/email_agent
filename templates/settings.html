After signing up, look for "API Keys" or "Integrations" in their settings/dashboard. Copy your API key.

3

Add to Environment File

Contact your administrator to add the API key to the .env file:

APOLLO_API_KEY=your_api_key_here
HUNTER_API_KEY=your_api_key_here
SNOV_API_KEY=your_api_key_here
FINDTHATLEAD_API_KEY=your_api_key_here

4

Restart the App

The administrator needs to restart the application for the new API keys to take effect.
